## What is AevonX?

AevonX is a **native macOS application** that gives you complete control over your Linux server fleet — directly from your Mac. No web panels running on your server, no Electron wrappers, no compromises.

Your credentials are encrypted with **AES-256 zero-knowledge architecture** — they never leave your device, and we literally cannot access your servers.

> *Server management shouldn't require a web browser running on your server.*

## Core Features

### Server Management
- **Multi-server fleet** — manage unlimited servers with real-time monitoring
- **Interactive SSH terminal** — with snippets, themes, and command history
- **File Manager** — full SFTP browser with drag-and-drop
- **Cron jobs** — visual scheduler with script editor

### Websites & Domains
- **Nginx & Apache** — virtual hosts, reverse proxy, configuration
- **SSL certificates** — Let's Encrypt auto-provisioning and renewal
- **Domain & DNS management** — records, zones, and propagation checks

### Databases
- **10 engine support** — MySQL, PostgreSQL, MongoDB, Redis, MariaDB, SQLite, Elasticsearch, CockroachDB, Cassandra, Valkey
- **Visual query console** — run queries, browse tables, manage users
- **Backup & restore** — scheduled and on-demand

### Docker & Containers
- **Container lifecycle** — create, start, stop, remove, logs, shell
- **Docker Compose** — multi-container orchestration
- **Templates** — one-click deployment of common stacks

### Security
- **7-layer defense system** — firewall, fail2ban, SSH hardening, audit, and more
- **130+ security tools** — comprehensive protection suite
- **Auto-whitelist** — AevonX automatically prevents your IP from being blocked

### App Runtimes
- **PHP-FPM, Node.js, Python, Composer** — version management and configuration
- **Git repositories** — clone, pull, push, branch management

### Plugin Ecosystem
- **Marketplace** — discover and install community plugins
- **Developer SDK** — 18 hook points, 15 component types
- **Build & publish** — create plugins for the AevonX ecosystem

## Tech Stack

| Layer | Technology |
|-------|-----------|
| **App** | Swift, SwiftUI, macOS 14+ |
| **Core Engine** | Go (hardened with Garble obfuscation) |
| **Transport** | SSH2 with ChaCha20-Poly1305 / AES-256-GCM |
| **Encryption** | AES-256 zero-knowledge (credentials never leave device) |
| **Backend** | Laravel, PostgreSQL, Cloudflare R2 |
| **Auth** | Ed25519 signed tokens, ECDH secure channels |
